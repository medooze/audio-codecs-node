# audio-codecs-node
Audio codecs for Media and RTMP servers

## Dependecies

- libopus
- libspeex
- libavcodec newer than https://github.com/FFmpeg/FFmpeg/commit/fbfee6adea6166ad87ef46124095abbf079f7bc3 for correct AAC decoding
